# Arelos Blog

This is the new blog, built with Jekyll.

# Writing a Post.
-Fork this Repo
-Make a new Markdown file **with** the liquid frontmatter and write.
-Save the file in this format: `YY-MM-DD-Post-Title.md`
-Submit a PR and I'll have a look :)
